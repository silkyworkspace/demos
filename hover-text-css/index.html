HTMLとCSSで実装

  • 猫

    大きな瞳でこちらを見つめるネコのアップ。繊細なヒゲと柔らかな毛並みが引き立つ。好奇心と少しの警戒が混ざった表情が愛らしい。

  • 犬

    白黒の毛並みが印象的なイヌのポートレート。片耳の立ち方が個性を感じさせる。まっすぐな視線に、誠実さと優しさがにじむ。

  • アヒル

    草地にたたずむアヒル。灰色の羽根の層とオレンジのくちばしが映える。首を伸ばした姿が凛として、静かな自然の空気を伝える。

  • 象

    モノクロの浜辺を悠然と歩くゾウ。大きな耳と深い皺、力強い鼻が存在感を放つ。波音が聞こえそうな静けさの中、堂々とした一歩が印象的。

  • ヒョウ

    木の幹に寄りかかり休むヒョウ。斑点模様としなやかな体つきが美しい。リラックスした姿でも瞳は鋭く、野性の緊張感が漂う。

  • パンダ

    丸い体のパンダがこちらを見上げる一枚。白黒の模様とふわふわの毛並みが愛嬌たっぷり。穏やかな雰囲気に癒やされる。
