④ タグ型マルチ選択

<select multiple> の代替として、選択肢をチップ（タグ）で管理するUIパターンです。 候補をクリックして追加、× で削除、テキスト入力でカスタムタグも追加できます。

⚠️ タグの状態はそのままではフォーム送信できません

タグの一覧は JavaScript の Set（内部的なデータ置き場）に保存されているだけで、<input> の value には入っていません。このままフォームを送信しても、サーバーにはタグの情報が届きません。

解決策： 画面には表示されない <input type="hidden"> を追加し、タグが追加・削除されるたびに JavaScript でカンマ区切り（例: "デザイン,CSS,JavaScript"）の値に更新します。

① ユーザーがタグを追加・削除する
② JavaScript が hiddenInput.value = "デザイン,CSS,JavaScript" のように hidden input の値を書き換える
③ フォーム送信時に tags=デザイン,CSS,JavaScript がサーバーへ届く
