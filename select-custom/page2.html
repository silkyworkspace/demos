② カスタムドロップダウン（完全制御）

ネイティブ <select> をやめ、div + button + JS で再実装します。 各 option に色ドット・サブテキスト・ホバーアニメーションなどを自由に付与できます。

⚠️ このドロップダウンはそのままではフォーム送信できません

ブラウザが <form> を送信するとき、<input>・<select>・<textarea> の値だけを自動的に集めます。 このドロップダウンは <div> と <button> で作られているため、ブラウザには「ただの飾り」と判断され、選択した値が無視されてしまいます。

解決策： 画面には表示されない <input type="hidden"> を追加し、選択するたびに JavaScript でその値を更新します。

① ユーザーが選択肢をクリックする
② JavaScript が hiddenInput.value = "published" のように hidden input の値を書き換える
③ フォーム送信時に status=published がサーバーへ届く

ステータス選択（フォーム送信対応版）

  • 公開中 一般ユーザーに公開されています
  • 下書き 保存済み・未公開の状態
  • レビュー中 承認待ちです
  • 非公開 アクセスを制限しています

送信されるデータ（リアルタイム確認）

status = （未選択）
✓ 送信されました！  

<input type="hidden" name="status"> は画面に表示されませんが、form 送信時にはしっかり値が含まれます。 選択するたびにデータプレビューが更新されるのを確認してみてください。
