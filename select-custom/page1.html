① ネイティブ select + CSS装飾

appearance: none でブラウザ標準UIを除去し、CSS だけでボーダー・背景・フォントを自由にスタイリングできます。矢印アイコンはSVGをインラインで重ねる方法が最も柔軟です。

基本スタイル（optgroup 付き）

appearance: none でブラウザ標準の矢印を消し、SVGを position: absolute で重ねています。 optgroup はブラウザが自動でグループラベルとインデントを生成します。

カラーバリエーション

Success
Error
Minimal

ボーダー色・背景色・テキスト色を変えるだけで success / error / minimal の3バリエーションを表現。 フォーカス時の box-shadow もそれぞれの色に合わせています。
