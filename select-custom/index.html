Select Box スタイリング デモ

HTMLのSelectボックスをCSSでスタイリングする4つのパターン

①

ネイティブ select + CSS装飾

appearance: none でブラウザ標準のUIを除去し、SVGアイコンとボーダーカラーで独自スタイル。optgroup で区切りも付与。

デモを見る → ②

カスタムドロップダウン（完全制御）

div + JS でドロップダウンを再実装。各 option に色ドット・サブテキスト・ホバーアニメーションを自由に付与できる。

デモを見る → ③

size属性で常に展開

size="N" でリスト表示に。選択状態は background + color で装飾（ただしブラウザ差異あり）。

デモを見る → ④

タグ型マルチ選択

select の代替として、タグをチップで管理するUIパターン。チップをクリックして追加、× で削除できる。
